# TMS Frontend

Vue 3 SPA for the Translation Management Service.

## Setup

```bash
npm install
npm run dev
```

## Configuration

Create a `.env` file:
```env
VITE_API_URL=http://localhost:8000/api/v1
VITE_API_TOKEN=your_api_token_here
```

## Features

- Translation management (CRUD)
- Locale management
- Tag management
- JSON export
- Dynamic language columns
- Token-based authentication

## Build

```bash
npm run build
```
